Setup Guide

ESC/POS (Generic Thermal Printers)

Works with most cheap thermal printers. Connection examples:

  • tcp://192.168.1.100:9100 - Network printer
  • usb://0x0416:0x5011 - USB printer (vendor:product)
  • file:///dev/usb/lp0 - Linux USB device

Brother QL Series

Popular label printers (QL-800, QL-820NWB). Connection examples:

  • tcp://192.168.1.100:9100 - Network (QL-820NWB)
  • usb://0x04f9:0x209c - USB connection

CUPS (System Printer)

Use any printer installed on the system. Connection is just the printer name:

  • Brother_QL-820NWB - CUPS printer name
  • HP_LaserJet - Any system printer
