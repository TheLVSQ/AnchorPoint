AnchorPoint API

Base URL: /api/v1/

OpenAPI schema: /api/schema/

  • GET /api/v1/ - API root
  • GET /api/v1/me/ - Authenticated user profile
  • /api/v1/people/ - People CRUD (staff)
  • /api/v1/households/ - Households CRUD + member actions (staff)
  • /api/v1/groups/ - Groups CRUD + member actions (staff)
  • /api/v1/events/ - Events list/detail public; write for staff
  • /api/v1/checkin/sessions/ - Check-in sessions list/detail

Authentication currently uses Django session auth.
